Тестовое задание для AQA стажёра в компании учи

Подготовка к работе
-------------------

Для начала работы убедитесь, что у вас установлены [node.js + npm](https://nodejs.org/en/download)

Установите зависимости проекта, выполнив команду:
```shell
npm ci
```

Запуск тестов
-------------

Запустите тесты с помощью команды:
```shell
npx playwright test
```


Что исправлено
---------------

- заменён общий селектор кнопки на точный `BUTTON_WRITE_TO_US = '[data-test="button_feedback_form"]'`.
- клики выполняются через `.first()` с ожиданием видимости, чтобы избежать strict-mode ошибок.
- методы получения списка популярных статей теперь ожидают появления элементов внутри `wrapper()` перед возвратом локаторов.


Дополнительный тест
-------------------

Реализован один дополнительный тест поиска внутри виджета — `search suggests password recovery`.
Кратко о тесте:
- Открывает главную страницу и закрывает попап cookies.
- Открывает виджет через POM.
- Вводит в поле поиска `забыл пароль` с помощью `fillSearch`.
- Считывает заголовки статей и проверяет, что среди них есть статья с ключевыми словами для восстановления пароля (вхождение `забыл` + `логин` или `забыл` + `пароль`).

Примечание по POM
-----------------

Для реализации этого теста были добавлены POM-методы `fillSearch(query)` и `getArticleTitles()` в `tests/widget.page.ts`.
